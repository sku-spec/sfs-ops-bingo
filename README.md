# SFS Ops Bingo

Self-serve bingo card for the SFS Ops Quarterly All Hands. Single-file HTML — open in a browser, enter a name, get a personalized card.

Each name produces a unique, deterministic card (same name → same card every time). 50-item pool, 5×5 grid with a free center, click to mark, BINGO detection across rows/columns/diagonals with confetti.

## Usage

Open `SFS-Ops-Bingo.html` in any browser, or host via GitHub Pages.
